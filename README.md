UNIKOM Mobile App (Flutter)

Aplikasi mobile sederhana berbasis Flutter yang dibuat untuk praktikum Pemrograman Mobile. Aplikasi ini digunakan untuk menampilkan informasi jadwal, berita, dan profil pengguna.

Fitur
1. Login sederhana (username dan password)
2. Dashboard (jadwal harian dan berita)
3. Halaman jadwal mahasiswa
4. Halaman berita
5. Halaman profil (lihat dan edit data, ganti foto, logout)

Teknologi
1. Flutter
2. Dart
3. Provider (state management)
4. Database lokal (SQLite)

Catatan
Menggunakan database lokal (offline)
Login masih sederhana (dummy)
Ditujukan untuk pembelajaran dasar Flutter
